Kerjaan Web Developer

Web Developer adalah orang yang bertugas mengembangkan sebuah website menjadi aplikasi utuh yang bisa digunakan oleh pengguna. Web Developer biasanya yang memikirkan bagaimana website tersebut dirancang dan digunakan.

Sekarang pekerjaan web developer semakin dipermudah dengan adanya bantuan dari framework. Setiap bahasa pemrograman memiliki framework yang dilengkapi dengan fitur-fitur dan dokumentasi

Dalam web programming, ada dua peran yang bisa dimasuki yaitu front-end dan back-end developer. Front-end Developer bertugas untuk merancang tampilan dan interaksi pada website. Sedangkan Back-end Developer bertugas untuk merancang sistem dan pengolahan data user pada website
